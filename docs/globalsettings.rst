Global Settings
===============

.. currentmodule:: pubrunner

PubRunner makes use a settings file (~/.pubrunner.settings.yml) for information about location to store files, upload settings and cluster usage. These can be set up using a wizard (as described below) or by manually editing the file.

Wizard
------

The wizard command (below) will provide prompts to change the various settings that PubRunner will use for all projects.

.. code-block:: shell

   pubrunner --wizard
